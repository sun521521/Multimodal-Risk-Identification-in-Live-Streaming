# Multimodal-Risk-Identification-in-Live-Streaming
2025年快手等直播平台频遭黑产攻击（如1.7万僵尸账号批量发布违规内容），现有风控策略依赖单一模态（如仅文本或图像）且规则僵化，导致：  漏判：复杂场景（如语音转文字后的隐晦黑话、图像内嵌恶意链接）难以识别。 误判/效率低：纯人工或简单模型审核时效性差（攻击发生后90分钟才拦截），且无法给出可解释的处置理由。 数据孤岛：用户行为、画像与内容特征割裂，无法关联分析（例如：高信用用户突然发布异常内容）。 项目目标：设计并验证一个基于大模型的多模态策略产品原型，实现从“单点规则拦截”到“端到端联合审判”的策略升级，输出风险类型、详细理由及处置建议，将平均响应时间从小时级降至秒级。
